# Gold Data — Trenzition V5 金标子集

同时满足四个条件的**高质量训练子集**：

1. **pKd 有效** (pkd_raw 非空)
2. **kcat 有效** (has_kcat=True)
3. **辅因子非空** (cofactors 有值)
4. **温度非默认** (来自 OED/BRENDA 匹配, ≠298.15K)

## 用途

此子集适用于 TST (Transition State Theory) 方案 A 训练。
每条记录都有完整的 pKd、kcat、辅因子和温度信息，
满足 Eyring 方程作为物理约束的全部必要条件。
